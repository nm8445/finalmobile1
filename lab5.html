Welcome to Lab 5

In my fifth lab for mobile development, I explored gesture handling and its critical role in creating responsive, intuitive, and accessible user experiences on touch-screen devices. Gestures like tapping, dragging, and scaling are essential for modern applications, with Flutter and React offering different approaches to implementation. Flutter uses the GestureDetector widget for simple and customizable gesture handling, while React relies on touchable components like TouchableOpacity. Flutter’s cross-platform support and intuitive widgets make it beginner-friendly for basic gestures, but advanced customization can be challenging and may add overhead. In my code examples, I implemented tap and swipe gestures to change UI elements, demonstrating Flutter’s flexibility. A real-world application of this concept is a photo gallery app, where users swipe to navigate images—a common and seamless interaction found in everyday mobile use.
