Welcome to Lab 2

In my second lab in XML and related technologies, I created a virtual event management system using XML to plan and manage events by inputting details, managing participants, and tracking schedules and resources. I designed the XML structure with a root element, events, and child elements like organization details (e.g., name, logo, website, and social media links), event attributes, and elements for hosts, participants, resources, and feedback. I incorporated optional features, such as virtual event components and marketing campaigns, to enhance creativity. The lab involved creating an XML tree diagram to represent the structure, defining a DTD to validate the document, writing a well-formed XML file with two fully populated events, and validating the XML against the DTD to ensure accuracy. Through this lab, I gained hands-on experience in working with XML syntax, attributes, and elements while learning how to create and validate structured data for practical applications.
